# Localization (EN/FR)

## Entry points
- English: `index.html`
- French: `index_fr.html`

## Rules
- Keep structure and interactions consistent between EN and FR.
- Translate visible UI text, aria-labels, and alt text.
- Verify `lang` attribute is correct on each HTML file.

## Content parity checklist
- Same number of sections/cards
- Same interactions (buttons, modals, accordion behavior)
- Same assets (or FR-specific equivalents) where required
