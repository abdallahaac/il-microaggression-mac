# Microaggressions Recognition (IL)

This repository contains the Interactive Learning (IL) package for **Reconnaître les microagressions envers les Autochtones**.

## What's inside
- `index.html` — English experience
- `index_fr.html` — French experience
- `assets/` — images, icons, and static assets
- `manifest.xml` — SCORM manifest (packaging + launch entry)
- `docs/` — technical documentation (this folder)

## Quick start (local)
Open the files directly in a browser for a quick review:
- `index.html`
- `index_fr.html`

> Note: Some LMS integrations behave differently than local viewing. See **docs/Brightspace.md** and **docs/SCORM.md**.

## Build/packaging (what to zip)
For distribution/publishing, include:
- `index.html`
- `index_fr.html`
- `manifest.xml`
- `assets/`
- `package.json` + lockfile (if used for tooling)
